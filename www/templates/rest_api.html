Faker UI is also available as REST. Just send a JSON object representing the data you want to generate and save the output to a file.

http://fakerui.pagodabox.com/data.{format}

Simple example

$ curl -d '[{"type":"firstName"},{"type":"lastName"}]' 'http://fakerui.pagodabox.com/data.sql?size=5'
INSERT INTO %table% VALUES("Paul", "Noel");
INSERT INTO %table% VALUES("Nicole", "Monnier");
INSERT INTO %table% VALUES("Patricia", "Vaillant");
INSERT INTO %table% VALUES("Adele", "Martins");
INSERT INTO %table% VALUES("Constance", "Mahe");

Adding titles

$ curl -d '[{"type":"firstName","title":"Prenom"},{"type":"lastName","title":"Nom"}]' 'http://fakerui.pagodabox.com/data.csv?size=5&titles=1'
